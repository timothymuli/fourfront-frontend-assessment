# Fourfront Frontend Assessment (HTML/CSS/JS + Bootstrap)

## What this is
A simple responsive page made with:
- HTML
- CSS
- JavaScript
- Bootstrap (CDN)

It tries to match a layout provided.

## Features / Requirements checklist
- Bootstrap grid used to center and size the layout
- Responsive for mobile/tablet/desktop
- JS Interactivity:
  - Click **Foundation Membership** or **Economy Membership** to show/hide the description
  - Simple email validation in the form

## How to run
1. Download or clone the repo
2. Open `index.html` using a browser (Chrome recommended)

No build steps needed.

## Notes
- The avatar uses an online placeholder image. You can replace it with your own image inside `assets/` and update the `<img src="">` in `index.html`.
